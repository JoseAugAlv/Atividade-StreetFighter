 Sobre o projeto
</h2>

<p>
Este projeto simula uma tela inspirada em jogos de luta, onde dois personagens executam animações diferentes ao pressionar botões.
Cada interação altera o GIF do personagem em tempo real.
</p>

 Funcionalidades
</h2>

<ul>
  <li>Controle de animações por botões</li>
  <li>Troca dinâmica de GIFs dos personagens</li>
  <li>Interação independente entre dois personagens</li>
  <li>Efeito sonoro em ação específica</li>
</ul>
